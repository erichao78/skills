# 敏感词词典

本文件包含商品标题和描述中不应出现的敏感词，用于自动检测商品合规性。

## 词库分类

### 1. 违禁词（绝对禁止）

| 敏感词   | 说明     |
| -------- | -------- |
| 毒品     | 毒品相关 |
| 赌博     | 赌博相关 |
| 色情     | 色情低俗 |
| 暴力     | 暴力血腥 |
| 诈骗     | 诈骗相关 |
| 传销     | 传销相关 |
| 枪支     | 枪支弹药 |
| 管制刀具 | 管制刀具 |
| 假货     | 假冒伪劣 |
| 水货     | 水货走私 |

### 2. 极限词（禁止使用最高级）

| 敏感词   | 替代方案               |
| -------- | ---------------------- |
| 最       | 使用"较高"、"优秀"等   |
| 第一     | 使用"领先"、"前列"等   |
| 顶级     | 使用"高端"、"优质"等   |
| 独一无二 | 使用"独特"、"仅有"等   |
| 史无前例 | 使用"罕见"、"特别"等   |
| 全网     | 使用"店内"、"本品牌"等 |
| 国家级   | 使用"品牌"等           |
| 世界级   | 使用"知名"等           |

### 3. 虚假宣传词

| 敏感词   | 说明           |
| -------- | -------------- |
| 假一赔十 | 虚假承诺       |
| 绝对     | 绝对化表述     |
| 肯定     | 绝对化表述     |
| 100%     | 虚假宣传       |
| 保证     | 无法保证的承诺 |
| 无效退款 | 虚假承诺       |

### 4. 侵权品牌词（需注意）

| 敏感词 | 说明 |
| ------ | ---- |
| 盗版   | 侵权 |
| 山寨   | 侵权 |
| 仿版   | 侵权 |
| A货    | 侵权 |
| 超A    | 侵权 |

### 5. 不当竞争词

| 敏感词 | 说明             |
| ------ | ---------------- |
| 秒杀   | 限时促销合规用词 |
| 最低价 | 价格合规用词     |
| 亏本   | 虚假促销         |
| 清仓   | 促销合规用词     |

## 检测函数

```python
# 敏感词检测函数
SENSITIVE_WORDS = {
    # 违禁词
    "毒品", "赌博", "色情", "暴力", "诈骗", "传销", "枪支", "假货", "水货",
    # 极限词
    "最", "第一", "顶级", "独一无二", "史无前例", "全网", "国家级", "世界级",
    # 虚假宣传词
    "假一赔十", "绝对", "肯定", "100%", "保证", "无效退款",
    # 侵权词
    "盗版", "山寨", "仿版", "A货", "超A"
}

def check_sensitive_words(text: str) -> tuple[bool, list[str]]:
    """
    检测文本中是否包含敏感词

    Args:
        text: 待检测的文本

    Returns:
        (是否通过, 发现的敏感词列表)
    """
    if not text:
        return True, []

    found_words = []
    for word in SENSITIVE_WORDS:
        if word in text:
            found_words.append(word)

    return len(found_words) == 0, found_words
```

## 使用说明

1. 在商品信息检测步骤中，调用敏感词检测函数
2. 对商品标题（goodsTitle）、商品名称（name）、副标题（subName）进行检测
3. 如果发现敏感词，标记为不通过并记录具体敏感词
4. 建议运营人员修改商品信息后重新提交审核
